Question-1: What are the different ways to select an element in the DOM?

DOM select করার ০৬ টি ধাপ আছে -


  1. ১. getElementById(): যা একটি নির্দিষ্ট id দিয়ে HTML Tag ধরতে হয়।
  2. ২. getElementsByClassName(): যা একটি নির্দিষ্ট class দিয়ে HTML Tag ধরতে হয়।
  3. ৩. getElementsByTagName(): যা একটি নির্দিষ্ট tag দিয়ে HTML Tag ধরতে হয়।
  4. ৪. querySelector(): এখানে CSS selector ব্যবহার করতে হয়।
  5. ৫. querySelectorAll(): এখানে CSS selector এর সব গুলো ব্যবহৃত হয়।
  6. ৬. getAttribute(): নির্দিষ্ট একটি attribute এর মান সংগ্রহ করতে ব্যবহার হয়।
Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, এবং textContent এই তিনটি প্রোপার্টি DOM-এ উপাদানের ভিতরের টেক্সট বা কনটেন্ট এক্সেস করার জন্য ব্যবহৃত হয়, তবে এগুলোর মধ্যে কিছু মৌলিক পার্থক্য রয়েছে। এখানে তাদের মধ্যে পার্থক্য ব্যাখ্যা করা হলো:
innerHTML:
- এটি একটি উপাদানের ভিতরের HTML কনটেন্ট (টেক্সট এবং HTML ট্যাগ সহ) ফেরত দেয় বা সেট করে।
- HTML ট্যাগসহ পুরো কনটেন্ট পরিবর্তন করতে বা পড়তে এটি ব্যবহার করা হয়।

innerText:
- এটি একটি উপাদানের ভিতরের টেক্সট কনটেন্টকে ফেরত দেয়, তবে শুধুমাত্র দৃশ্যমান টেক্সট, অর্থাৎ CSS দ্বারা hide করা টেক্সট বা hidden উপাদান বাদ দিয়ে।
- innerText সাধারণত টেক্সটের জন্য ব্যবহৃত হয়, HTML ট্যাগসহ কনটেন্ট নয়।

textContent:
- এটি একটি উপাদানের ভিতরের সব টেক্সট কনটেন্ট ফেরত দেয়, HTML ট্যাগগুলো বাদ দিয়ে।
- এটি ব্রাউজারের স্টাইলিং বা রেন্ডারিং প্রভাব ছাড়াই সব টেক্সট ফেরত দেয়।

Question-3: What is event delegation in the DOM?

event delegation হলো একটি টেকনিক যেখানে parent উপাদানে event listener এ যোগ করা হয়, পরিবর্তে প্রতিটি child উপাদানে আলাদাভাবে event listener যোগ করা হয়। এটি মূলত even bubble এর মাধ্যমে কাজ করে।

Question-4: What is event bubbling in the DOM?

event bubbling হলো একটি প্রক্রিয়া যেখানে একটি event প্রথমে ঐ উপাদানে ঘটতে থাকে যেটিতে event শুরু হয়েছে, তারপর তা parent উপাদানে চলে যায় এবং এরকম ভাবে উপরের দিকে bubbling হয়ে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript ব্যবহার করে নতুন element তৈরি করতে document.createElement() method ব্যবহার করা হয়। তারপর, সেই element DOM-এ যোগ করার জন্য appendChild() method ব্যবহার করা হয়। element মুছে ফেলতে removeChild() মেথড ব্যবহার করা হয়। এভাবে, HTML উপাদানগুলোর সাথে interact করে নতুন element তৈরি, যোগ এবং মুছে ফেলতে পারা যায়।
